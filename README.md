# Py & Cobol Practicals

This repository contains practical exercises and sample programs for learning Python and COBOL.

## Purpose
These files are intended for educational and practical learning purposes only. They are designed to help students and learners understand basic programming concepts, coding structure, and problem-solving approaches.

## About First Advantage Consulting Ltd (FAC)
These materials are prepared for learning and training support by First Advantage Consulting Ltd (FAC).

Website: https://academy.firstadvantageconsulting.com/

## What is included
- Python practice files covering topics such as:
  - classes and objects
  - inheritance and polymorphism
  - exceptions
  - input validation
  - file handling
  - simple business logic

- COBOL practice programs covering basic syntax and simple business-oriented examples.

## Notes
- These programs are meant for study, practice, and skill development.
- They are not intended for production deployment without further review and testing.
- Learners are encouraged to modify and experiment with the code to improve understanding.
