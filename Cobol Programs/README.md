# COBOL Programs

This folder contains simple COBOL practice programs for learning the basics of COBOL programming.

## Learning Purpose
These programs are created for educational and practical use only. They are intended to help learners understand COBOL structure, syntax, and common programming concepts.

## About First Advantage Consulting Ltd (FAC)
These learning materials are provided for training purposes by First Advantage Consulting Ltd (FAC).

Website: https://academy.firstadvantageconsulting.com/

## Program Examples
- GREETING.cob: a basic greeting program
- STUDENT.cob: a simple student-related example
- PATIENT.cob: a patient record example
- ASSIGNMENT1.cob and ASSIGNMENT2.cob: practice exercises

## Notes
- Use these files to learn COBOL fundamentals.
- Feel free to explore, edit, and test them in a suitable COBOL compiler environment.
- These files are meant for learning and practical study, not for live business use without proper review.
